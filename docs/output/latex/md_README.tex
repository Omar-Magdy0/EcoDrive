\hypertarget{md_README_autotoc_md17}{}\doxysubsection{\texorpdfstring{🧠 Overview}{🧠 Overview}}\label{md_README_autotoc_md17}
Eco\+Drive implements a range of hardware and software solutions for electric drive systems. ~\newline
 It includes\+:


\begin{DoxyItemize}
\item Custom inverters (multi-\/phase and bidirectional support)
\item Firmware for Trapezoidal , FOC, SPWM, SVM, and advanced motor control
\item Modular controllers and drivers
\item Fully documented software and hardware stacks
\item Design iterations built for real-\/world racing
\end{DoxyItemize}
