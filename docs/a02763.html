Detailed Description

STM32F4 MC3P 3phase motor-control driver API.

Author
Omar Magdy

This module configures TIM1, ADC1, and DMA2 to provide:

  • PWM generation for 3-phase bridge control (trap or SVM)
  • Injected ADC sampling synchronized to PWM for phase/current sensing
  • Background ADC scanning via DMA for slow-monitoring channels

Timing is driven by TIM1 center-aligned PWM. ADC injected conversions are triggered from TIM1 TRGO (CH4 reference), while regular conversions use DMA in circular double-buffer mode.
